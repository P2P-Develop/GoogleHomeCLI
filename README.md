# GoogleHomeCLI
ぐーぐるほーむ(ChromeCast)を、CLIで動かすやつ
Youtubeの動画も流せる！


## 必要なもの

+ ほんたい

+ Python 3.x

+ PyChromeCast
` pip install pychromecast `

## こまんど

+ use <デバイスIDまたは名前>
デバイスを選択するよ！  
起動すると、最初に認識したデバイスが自動選択されるよ！
+ play <動画ファイル直リンクまたはYouTubeリンク>
曲(動画)が再生されるよ！
+ status
ステータスがひょうじされるよ！
+ kill
プロセスを抹消するよ！
+ list
認識しているデバイスリストを表示するよ！
+ reconnect
デバイスを再認識させるよ！
+ exit
抜け出すよ！
