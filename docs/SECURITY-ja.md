## 脆弱性を発見したら

もし脆弱性を発見したら、\[security\] ラベルを付けて [GitHub Issues](https://github.com/P2P-Devleop/GoogleHomeCLI/issues) にて報告してください。
脆弱性が検証・発見され次第セキュリティー草案で Open し、修正します。
